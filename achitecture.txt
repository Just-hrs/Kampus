# KAMPUS / STUDENTOS — MASTER PROJECT SNAPSHOT
pnpm config set registry https://registry.npmjs.org
## PROJECT IDENTITY

Kampus (formerly StudentOS) is a mobile-native, offline-first college lifestyle OS built using React + Vite + Capacitor Android.

Core philosophy:

* not a productivity app
* not a generic attendance tracker
* intended as a “daily college companion”
* combines academics, attendance, grades, expenses, humor, anxiety culture, mini-games, and insights
* optimized primarily for Android phones

The app is installable as an APK using Capacitor.

## Current Major Milestone

Android rendering corruption/glitching was mostly fixed.

Root cause:

* heavy glassmorphism + Framer Motion + Android WebView GPU compositing

Critical fix:
File:
src/core/components/Surface.tsx

Changed default:
variant = "glass"

to:
variant = "flat"

Result:

* UI became stable
* glitch accumulation mostly disappeared
* dashboard stability improved
* Android WebView rendering corruption reduced heavily

Important:

* Keep flat surfaces as default on Android
* glass effects should remain minimal and isolated

# IMPORTANT ANDROID NOTES

Android WebView performance is a major constraint.

Avoid excessive:

* backdrop blur
* glassmorphism
* huge shadows
* nested opacity
* animated translucent layers
* heavy GPU compositing

Framer Motion + glass effects caused major instability earlier.

Flat UI is significantly more stable.

# IMPORTANT CONTEXT FOR FUTURE CHATS

If debugging Android:

* assume WebView compositor limitations first
* prefer flat UI
* avoid blaming routing/store immediately

If debugging semester issues:

* check shared Zustand state
* especially `activeSemesterId`

If debugging persistence:

* hydration is async
* app depends heavily on IndexedDB restore

If debugging UI:

* Surface.tsx is extremely important
* many rendering behaviors originate there

If debugging navigation:

* AppShell + TanStack Router control most flow

If adding analytics/telemetry:

* app is currently fully offline-first
* no backend yet
* no cloud sync yet
* no authentication system yet
